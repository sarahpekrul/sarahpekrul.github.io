Lists Usage
Unordered List An unordered list is a list of items that are not arranged in a particular order, like a collection, starting with <ul> and ending with </ul>. The default setting is usually bullet points.
Ordered List An ordered list is a list given in a specific order, like a recipe, starting with <ol> and ending with </ol> After the opening tag and before the closing tag, you need to insert a <li> and </li>. The default setting is usually numbered.
Definition List A list that gives a word and a matching definition. The <li> tags aren't neccessary here. Each list is opened with <dl> and closes with </dl>. There are two columns in this list, the terms <dt> and the definitions <dd>. Each item should be opened and closed with the correct tag identifying whether it is a term or definition.
